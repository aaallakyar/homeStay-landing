# HomeStay - Сервис аренды жилья для отпуска

## Тема проекта
Платформа для поиска и аренды жилья на короткий срок (от 1 дня до нескольких недель) с гарантией качества и мгновенным бронированием.

## Использованный фреймворк
**Fomantic UI** (v2.9.3) - CSS-фреймворк на основе Semantic UI, ориентированный на семантический HTML и человеко-понятные классы.
